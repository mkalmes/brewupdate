h1. brewupdate

<code>brewupdate</code> is a "launchd agent":http://developer.apple.com/library/mac/#technotes/tn2083/_index.html to update "homebrew":https://github.com/mxcl/homebrew/ formulae automaticly.

h2. How to install it?

To install <code>brewupdate</code>, copy the plist to <code>~/Library/LaunchAgents</code> and run the command <code>launchctl load ~/Library/LaunchAgents/brewupdate.plist</code> to load the agent into your user launchd manager.

<code>brewupdate</code> will update the homebrew formulae every day at 11 AM (local time).
